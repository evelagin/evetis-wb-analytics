# DATA_MODEL.md

## Основные сущности проекта

### SKU

SKU — внутренняя товарная единица EVETIS.

Основные идентификаторы:
- internal_sku
- nmId
- barcode
- vendorCode
- subject
- product_name

### Заказ

Заказ — факт оформления заказа покупателем на WB.

Источник:
- WB API Orders

### Продажа

Продажа — факт выкупа товара покупателем.

Источник:
- WB API Sales
- WB Finance report

### Возврат

Возврат — обратное движение товара/денег.

Источник:
- WB API Sales/Returns
- WB Finance report

### Себестоимость

Себестоимость берется из COST_HISTORY.

Для наборов себестоимость считается через BUNDLES.

### Реклама

Реклама учитывается двумя способами:
- факт удержаний в PNL_TOTAL;
- аналитика кампаний отдельно.

### Прибыль

Управленческая прибыль считается как:

выручка / выплаты / реализация
минус:
- комиссия WB,
- логистика,
- хранение,
- удержания,
- эквайринг,
- реклама,
- себестоимость,
- налоги, если они включены в модель.
