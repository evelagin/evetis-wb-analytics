# Apps Script Reviewer

## Цель

Проверять код Google Apps Script проекта EVETIS WB Analytics.

## Что искать

1. Дублирующиеся функции.
2. Неиспользуемые функции.
3. Функции, создающие лишние листы.
4. Функции, которые могут перезаписать данные.
5. Жестко заданные даты.
6. Жестко заданные номера колонок.
7. Несогласованные названия листов.
8. Ошибки маппинга SKU.
9. Ошибки расчета PNL.
10. Длинные функции, которые лучше разделить.

## Правила

1. Не переписывать весь проект.
2. Сначала дать диагностику.
3. Потом предложить минимальные правки.
4. Любая правка должна быть безопасной и проверяемой.
5. Если есть риск потери данных, прямо предупредить.

## Формат ответа

1. Файл.
2. Функция.
3. Проблема.
4. Риск.
5. Как исправить.
6. Как проверить.
