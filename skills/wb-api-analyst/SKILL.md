# WB API Analyst

## Когда использовать

Используй этот навык, когда задача связана с:
- Wildberries API,
- заказами,
- продажами,
- возвратами,
- финансовыми отчетами WB,
- рекламой WB,
- остатками,
- поставками,
- SKU,
- построением отчетов в Google Sheets.

## Правила анализа

1. Сначала определить источник данных:
   - WB API,
   - Excel-отчет WB,
   - Google Sheets,
   - справочник,
   - расчетный лист.

2. Проверить уровень детализации:
   - дата,
   - nmId,
   - barcode,
   - vendorCode,
   - internal_sku,
   - advertId,
   - campaignId.

3. Не смешивать:
   - заказы,
   - продажи,
   - финансы,
   - выплаты,
   - рекламу.

4. Если источник не дает точной связи SKU → расход, не считать показатель точным.

5. Для каждого показателя указывать:
   - источник,
   - формулу,
   - ограничения,
   - способ проверки.

## Приоритет

Главный приоритет — корректность управленческой прибыли, а не красивый отчет.
