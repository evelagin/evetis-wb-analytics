# Google Sheets Architect

## Цель

Создавать простую и понятную структуру Google Sheets для EVETIS WB Analytics.

## Правила

1. Не создавать новые листы без необходимости.
2. Не создавать листы-дубли.
3. Не смешивать RAW-данные и расчетные данные.
4. Заголовки должны быть на русском языке.
5. Статусы должны быть на русском языке.
6. Лист должен быть понятен пользователю без изучения кода.
7. Важные отчеты должны иметь:
   - дату обновления,
   - источник данных,
   - статус,
   - контрольные суммы.

## Рекомендуемые типы листов

1. RAW-листы.
2. Справочники.
3. Расчетные витрины.
4. Финальные отчеты.
5. Dashboard.

## Перед изменением структуры

Всегда сначала ответь:
1. какой лист меняется;
2. какие колонки добавляются;
3. какие колонки удаляются;
4. какие формулы затрагиваются;
5. как проверить результат.
